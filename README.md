# traffic-stuck
App in Html/Css/JQuery using plugin JQuery UI Draggable to drag the red car out of the traffic jam

# HOW TO INSTALL (using WampServer)

1) Start your Wamp </br>
2) Move the directory "rushHour" into your Wamp "www" directory </br>
3) Go to  -> http://localhost/rushHour/webroot/ </br>
4) Enjoy !!! </br>
